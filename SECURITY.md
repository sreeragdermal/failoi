# Security Policy

## Supported Versions

The following versions of FAILOI receive security updates:

| Version | Supported |
| :--- | :--- |
| v1.0.0-beta | ✅ Yes |
| < v1.0.0 | ❌ No |

## Reporting a Vulnerability

We take the security of FAILOI seriously. If you find a security vulnerability, please do **not** report it publicly via GitHub issues. Instead, follow these steps:

1. Draft a detailed explanation of the vulnerability, including step-by-step reproduction instructions or code snippets.
2. Email the report to **security@yourdomain.com**.
3. We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.

Please give us reasonable time to investigate and patch the vulnerability before public disclosure.
